functions.php của unikey:
    $requires = [
        '/base/tailwindcss/set-up.php',
        '/base/helpers/common.php',
        '/base/ux-builder/set-up.php', // nếu dùng ux builder shortcode, thêm lại các ux builder shortcode cần dùng
    ];

    foreach ($requires as $file) {
        require_once get_stylesheet_directory() . $file;
    }
    require_files('/base/shortcodes/');
ở footer nhớ <?php wp_footer(); ?> để load js của base/tailwindcss/set-up.php
copy thư mục base cho child theme
copy các file cho child theme:
    package.json
    postcss.config.mjs
    tailwind.config.js
    vite.config.js
npm install
active child theme cần dev
truy cập thư mục child theme:
    cấu hình wp-config.php > WP_DEBUG > [true] nếu đang dev ở local dùng [npm run dev] / [false] nếu cần build cho prod dùng [npm run build]
    cấu hình wp-config.php > WP_DEBUG_SQL_QUERY = true nếu cần debug db query
